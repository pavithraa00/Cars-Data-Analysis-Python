🚗 **Cars Data Analysis using Python & Pandas**

I recently completed another hands-on data analysis project on Cars Dataset, implementing it step by step.

---

🔍 **Project Overview**
This project focuses on performing Exploratory Data Analysis (EDA) on a real-world Cars dataset using Python. The goal was to understand the dataset, clean it, and extract meaningful insights using powerful data analysis techniques provided by Pandas.

---

🎯 **Objectives**

* Explore and understand dataset structure
* Perform data cleaning and preprocessing
* Analyze data using Pandas operations
* Solve real-world analytical questions

---

📂 **Dataset**

The dataset contains time-series weather data with the following features:

* Make
* Model
* Type
* Origin
* DriveTrain
* MSRP
* Invoice
* EngineSize
* Cylinders
* Horsepower
* MPG_City
* MPG_Highway
* Weight
* Wheelbase
* Length

---

🛠️ **Technologies Used**

* Python 
* Pandas
* NumPy
* Google Colab

---

📂 **Key Tasks Performed**

✔️ **Data Exploration**

* Viewed dataset structure and summary
* Checked data types and null values

✔️ **Data Cleaning**

* Handled missing values
* Converted data types where necessary

✔️ **Data Analysis**

* Applied filtering using conditions
* Used `groupby()` for aggregation
* Calculated statistical measures (mean, max, min)

✔️ **Query Solving**

* Extracted specific insights from dataset
* Applied multiple conditions (AND / OR filtering)
* Analyzed patterns and trends
